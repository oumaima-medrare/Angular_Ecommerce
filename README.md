## Development server
**-> Install JSON Server :** <br />
`npm install --save-dev json-server`  <br />
**-> Run Json Server :** <br />
`npm run api` <br />
**-> Run dev server :**  <br />
`ng serve`. <br />

Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files. <br />
